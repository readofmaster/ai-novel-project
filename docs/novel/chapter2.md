# 第2章：庭いじりの夜明け

「鈴木め、やりやがったな」

朝焼けの中、佐藤はコーヒーカップを握りしめたまま、窓の外に釘付けになった。隣家との境界線に置かれた鈴木の庭が、昨日までとは比べ物にならないほど洗練されている。昨日までただの茶色い土だった場所に、幾何学模様の砂利が敷かれ、さらには異国情緒あふれる奇妙な形の低木が、まるで誇らしげに背筋を伸ばしていた。

「景観整備なら、俺だって負けていられない……！」

佐藤の闘志に火がついた。彼は急いでホームセンターへ直行し、ありったけの園芸用レンガと、最高級の芝生の種を買い占めた。帰宅するなり、彼は庭へ飛び出す。スコップを振るう腕に力がこもる。単なる草むしりではない。これは防衛戦であり、威信をかけたテリトリー争いだ。

佐藤は庭の角に、ネット通販で衝動買いした「北欧風のガーデンライト」を突き立てた。鈴木の低木が持つ自然の美しさに対し、こちらは無機質な文明の光で対抗する戦略だ。さらに、庭の縁取りには、一つ一つ丁寧に磨き上げた白い玉砂利を配置する。その姿は、まるで陣地を広げる将軍のようだった。

数時間後、汗だくになった佐藤が腰を伸ばすと、ちょうど隣からも「ふぅ」という吐息が聞こえた。見れば、鈴木もまた泥だらけの顔で、こちらをじっと見つめている。二人の視線が空中で交差した。

「いい趣味ですね、佐藤さん」
「そちらも、なかなか……派手な低木ですね」

お互いに引きつった笑顔を浮かべる。佐藤の心の中には、勝利の確信と、次なる一手をどう打つかという新たな策謀が渦巻いていた。明日の朝、この庭はまた別の姿に生まれ変わっているはずだ。
静かな住宅街の朝に、園芸という名の静かなる狂騒曲（ラプソディ）が幕を開けた。
