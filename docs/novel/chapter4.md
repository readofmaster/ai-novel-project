# 第4章：庭石の逆襲

夜空に浮かぶのは、鈴木の庭から放たれる青白い発光植物の怪光線ではない。佐藤の庭で、重々しくも軽やかに躍動する影だ。

「……動いた」

佐藤は震える手で懐中電灯を握りしめ、窓から庭を見下ろしていた。彼が最新の園芸ショップで、「意志を持つ石庭キット」として購入した黒い御影石の群れ。当初はただの置物だと思っていたが、鈴木の光る植木が夜な夜な庭を照らし始めてから、状況は一変した。

石たちは、鈴木の植木が夜な夜な放出する謎のエネルギーを吸収し、深夜二時を回ると決まって「行進」を始めるのだ。それはまるで、社交ダンスの練習をしているかのように優雅で、かつ攻撃的だった。

「いいか、配置は完璧だ。あいつがこっちに光線を向けてくるなら、こっちは物理的な圧で対抗する！」

佐藤が庭のスイッチを入れると、庭石たちが一斉に鈴木の敷地との境界線へとなだれ込む。鈴木の植木が放つ「ブーン」という電子音のような羽音に対し、庭石たちは「コツ、コツ、コツ」と重厚なリズムを刻み、完璧なフォーメーションで防壁を築き上げた。

翌朝。カーテンを開けた佐藤は、思わずコーヒーを吹き出した。
昨晩の行進の結果、佐藤の庭石たちは、鈴木の光る植木を完全に包囲し、さらにそれらを「花びら」のように見立てて、巨大な「花の形をした石のオブジェ」へと再構築していたのだ。

そこへ、鈴木が寝癖のついた頭で現れる。
「おい、佐藤さん。うちの植木が……勝手に石と合体して、妙なダンスを踊ってるんだが？」
「知らんよ。……いや、これは『庭の意思』だ。お互いの庭が、勝手に高め合っているのさ」

佐藤は誇らしげに胸を張った。鈴木は呆れながらも、その石と植物が織りなす奇怪な造形美を、どこか感心したような目で眺めていた。住宅街の平和は、今や完全に狂った芸術の領域に突入していた。
