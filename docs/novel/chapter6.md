# 第6章：境界線、消滅のお知らせ

「……これ、どういう状況ですか」

配送業者の青年の震える声が、住宅街に響き渡った。佐藤と鈴木は、それぞれの手から剪定バサミを置き、顔を見合わせた。

佐藤の庭にあった「動く石像」と、鈴木の庭の「発光する植木」が、互いの領土を浸食し合うように絡まり合っている。石像は光を浴びて宝石のように脈動し、植木の根は石像の隙間を縫って、まるで意思を持つかのように佐藤の敷地へと侵攻していた。

「いや、鈴木さん。君の木が、うちの石像に巻き付いているんだが」
「佐藤さん、それは違う。あなたの石像が、私の木に栄養を吸わせるように誘っているんだよ。これはもう、共同プロジェクトだ」

二人は確信していた。互いの個性がぶつかり合うことで、単なる「庭」という枠を超えた、有機的な融合体が生まれようとしていることを。もはや境界線など無意味だった。境界線上の柵は、すでに植物のツタと石像の破片に覆われ、跡形もなくなっている。

二人は無言で頷き合い、塀を取り払った。隣家との壁が崩れ去り、二つの庭が一つに繋がった瞬間、庭全体が「ギュイン」という奇妙な音を立てて波打った。

「見てくれ、佐藤さん。うちの光る花が、君の石像の足元に咲き誇っている！」
「ああ、そして私の石像が、君の庭に独自の生態系を作り始めているぞ！」

住宅街の住民たちは、窓からその光景を呆然と眺めていた。そこにはもはや、かつての平穏な住宅の面影はない。一つの広大な、奇想天外な異空間が出現していたのである。二人のエゴとエゴが溶け合い、住宅街という名のリビングを、怪しくも壮大なジャングルへと変貌させていた。

「さあ、次の計画だ。次は……池を作らないか？」

二人の狂気じみた情熱は、とどまるところを知らない。
