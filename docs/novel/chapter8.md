# 第8章：グリーン・パニック！ 住宅街はボタニカル・パラダイス

　その日の朝、町内会の回覧板は、もはや植物に飲み込まれて戻ってこなかった。

　佐藤と鈴木の庭から溢れ出した「究極の園芸」は、もはや個人の趣味の範疇を超えていた。佐藤が植えた自己増殖型のドリル・チューリップが隣家の屋根を貫通し、鈴木が開発した「意思を持つ蔦」が電柱を伝って隣の町内まで侵食を始めている。

「鈴木さん、見てください！ 我が家の庭から出土した古代種の光るシダが、歩道のコンクリートを突き破って咲き誇っていますよ！」
「佐藤さん！ なんて素晴らしい！ 我が家の庭の土壌センサーが検知したところ、この住宅街そのものが『巨大なプランター』と認識されたようです！」

　二人は興奮気味にハイタッチを交わした。もはや玄関ドアを開ければ、そこはリビングではなく、密林の入り口だ。近隣住民は困惑しつつも、もはや庭の美しさに逆らう気力を失っていた。通りは、見たこともない極彩色の花々で埋め尽くされ、夜になると住宅街全体が怪しく、しかし幻想的に発光する。

　郵便配達員は、空を飛ぶ蔦の足場を軽やかに跳ねて郵便物を届けるようになり、タクシーの運転手は「あの緑のトンネルの三軒目」と道案内をするようになった。住宅街は完全にボタニカル・パラダイスと化した。

　佐藤は満足げに、自らが育てた「飛行する水蓮」に腰掛けた。「鈴木さん、私たちはついに地球を庭にしたのです」
「ええ、佐藤さん。明日はさらにその先へ……山脈をもデザインしましょう」

　もはや誰にも止められない。狂気じみた園芸の連鎖は、止まることを知らないのだった。
