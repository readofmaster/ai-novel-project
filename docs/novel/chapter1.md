# 第1章：静かなる開戦

佐藤が愛する庭には、これまで「調和」という名の平和が流れていた。季節の花を植え、毎週芝を刈り、隣家との境界線には控えめなラティスを置く。それだけで十分、美しい住宅街の景観は保たれていたはずだった。

あいつ――隣の鈴木が、あの「奇妙な青い花」を連れ帰ってくるまでは。

土曜の朝、コーヒー片手に縁側へ出た佐藤は、思わず噴き出しそうになった。鈴木の庭の中央に、見たこともない品種が鎮座していたのだ。花弁は毒々しいほどに鮮やかなネオンブルーで、あろうことか、日差しを浴びるたびに微かに金属的な光沢を放っている。

「……なんだ、ありゃ」

佐藤は目をこすった。合成着色料で染めたのかと疑うレベルだ。しかも、その花は不自然なまでに背が高く、佐藤の庭を睥睨（へいげい）するように咲き誇っている。隣家の庭の質感が、一瞬にしてそこだけ異界のそれに塗り替えられたような感覚。

鈴木は、ジョウロを片手に満足げな笑みを浮かべていた。やつは佐藤の視線に気づくと、わざとらしく花を愛でるような仕草をし、「おはよう、佐藤さん。このブルー・インフェルノ、なかなかでしょう？」と爽やかに挨拶してくる。

ブルー・インフェルノ。直訳すれば『青い地獄』だ。なんて不吉で、なんて挑発的なネーミングだ。

佐藤の心の中で、何かがパチンと弾けた。これまで培ってきた「周囲との調和」という美徳が、鈴木の植えた異形の植物によって真っ向から否定されたのだ。これは趣味の問題ではない。生存競争だ。

「……ああ、確かに。目が潰れそうなくらい、個性的で『面白い』庭だね」

佐藤は必死で引きつる頬を隠しながら、そう言い返した。その瞬間、佐藤の中で静かな開戦のゴングが鳴り響く。平和な週末が、終わりを告げた。
