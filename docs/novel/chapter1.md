# 第1章：静かなる開戦（序章）

ある平日の夕方、佐藤の平和な日常は、隣人・鈴木の庭によって終わりを告げた。
正確には、鈴木が何気なく植えた、見たこともない奇妙な花——いや、あれは本当に花なのだろうか？——のせいだ。
その花は、夕闇の中で妖しげに青白く発光し、まるで「俺を見ろ、俺を崇めよ」と主張しているかのように、佐藤の家のリビングを侵食していた。

「なんだよ、あれ……」

佐藤は思わず窓を閉めた。コーヒーカップを持つ手がかすかに震える。
佐藤にとって、庭はただの「雑草を抜くための場所」だった。休日の朝、少しばかり草を抜き、近所の人と軽く挨拶を交わす。そんな穏やかなルーチンこそが、彼の誇りだったのだ。
しかし、隣の鈴木の庭は違った。最近、鈴木は庭いじりに異様な情熱を注ぎ始めている。朝の５時から土をいじり、夜は怪しげなライトアップを施す。
そして今夜だ。あの発光する花。

「あれには負けられないな」

佐藤の中で、何かがカチリと音を立てて切り替わった。
それは嫉妬か、あるいは生存本能か。
佐藤は立ち上がると、納屋から埃を被っていた園芸ハサミを引っ張り出した。
隣の庭に負けない、いや、隣の庭を圧倒するような、最高の「何か」を植えてやる。
静かな住宅街に、静かだが確実に、開戦の狼煙が上がった瞬間だった。

そう決意した瞬間、佐藤の瞳には闘志の炎が宿った。彼はネット通販で「成長速度 日本一」という胡散臭いキーワードを検索し、無造作にカートへぶち込んだ。
「待っていろよ、鈴木。お前の光る花なんか、俺の庭の圧倒的ボリュームの前に屈服させてやる……！」
佐藤は夜の闇に向かって拳を突き上げた。隣の家から「ふふっ」という不気味な笑い声が聞こえたような気がしたが、気にしないことにした。
明日からは、仕事どころではない。園芸こそが、これからの彼の人生における最優先事項となるのだ。
さあ、終わりのない庭いじり戦争の幕開けだ。佐藤の庭は、果たして鈴木の庭を凌駕できるのか。

## 第1章：静かなる開戦

佐藤が愛する庭には、これまで「調和」という名の平和が流れていた。季節の花を植え、毎週芝を刈り、隣家との境界線には控えめなラティスを置く。それだけで十分、美しい住宅街の景観は保たれていたはずだった。

あいつ――隣の鈴木が、あの「奇妙な青い花」を連れ帰ってくるまでは。

土曜の朝、コーヒー片手に縁側へ出た佐藤は、思わず噴き出しそうになった。鈴木の庭の中央に、見たこともない品種が鎮座していたのだ。花弁は毒々しいほどに鮮やかなネオンブルーで、あろうことか、日差しを浴びるたびに微かに金属的な光沢を放っている。

「……なんだ、ありゃ」

佐藤は目をこすった。合成着色料で染めたのかと疑うレベルだ。しかも、その花は不自然なまでに背が高く、佐藤の庭を睥睨（へいげい）するように咲き誇っている。隣家の庭の質感が、一瞬にしてそこだけ異界のそれに塗り替えられたような感覚。

鈴木は、ジョウロを片手に満足げな笑みを浮かべていた。やつは佐藤の視線に気づくと、わざとらしく花を愛でるような仕草をし、「おはよう、佐藤さん。このブルー・インフェルノ、なかなかでしょう？」と爽やかに挨拶してくる。

ブルー・インフェルノ。直訳すれば『青い地獄』だ。なんて不吉で、なんて挑発的なネーミングだ。

佐藤の心の中で、何かがパチンと弾けた。これまで培ってきた「周囲との調和」という美徳が、鈴木の植えた異形の植物によって真っ向から否定されたのだ。これは趣味の問題ではない。生存競争だ。

「……ああ、確かに。目が潰れそうなくらい、個性的で『面白い』庭だね」

佐藤は必死で引きつる頬を隠しながら、そう言い返した。その瞬間、佐藤の中で静かな開戦のゴングが鳴り響く。平和な週末が、終わりを告げた。
