# 登場人物紹介

本作に登場する主要キャラクターの紹介です。

## 名前の選定意図について
本作の主人公「佐藤」と、その隣人「鈴木」は、日本で最もありふれた代表的な苗字です。
どこにでもある日常的で平凡な住宅街を舞台に、この二人が極めてエキセントリックでSFチックな「庭いじりバトル」を繰り広げるという【日常と非日常の極端なギャップ】をコミカルに描くために、あえて誰にでも馴染みのある分かりやすい名前にしています。

## 佐藤（さとう）
*   **特徴**: 負けず嫌いな常識人。鈴木の奇抜な庭いじりに引きずられるように張り合い、最新の庭園ガジェットや動く庭石を導入する。
*   **庭いじりの目的**: 我が家のプライドとテリトリーを守る防衛戦。

## 鈴木（すずき）
*   **特徴**: マイペースでエキセントリック。園芸店の「裏ルート」から電気ウナギと交配した光る植木などを仕入れてくる行動派。
*   **庭いじりの目的**: 独創的な美学の追求と、佐藤との「共演」を楽しむこと。
