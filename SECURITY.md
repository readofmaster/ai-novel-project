# Security Policy

## Reporting a Vulnerability / 脆弱性の報告について

If you find a security vulnerability, please do NOT open a public issue. 
Instead, please report it privately using the **"Report a vulnerability"** button found under the **Security** tab of this repository.

本リポジトリでセキュリティ上の問題（脆弱性）を発見した場合は、公開のIssueには投稿せず、
リポジトリの **「Security」タブ** にある **「Report a vulnerability」** ボタンから非公開でご報告をお願いいたします。

Thank you for helping keep this project safe! / ご協力ありがとうございます！
